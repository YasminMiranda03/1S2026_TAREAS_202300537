# Errores Léxicos Detectados

Durante el análisis del código fuente se detectaron varios errores léxicos.
Un error léxico ocurre cuando una secuencia de caracteres no corresponde a ningún patrón válido del lenguaje.

## Lista de errores

| Lexema    | Descripción del error                                     |
| --------- | --------------------------------------------------------- |
| 123numero | Un identificador no puede iniciar con un número           |
| valor#    | El carácter `#` no es permitido dentro de identificadores |
| 123numero | Repetición del mismo error en otra parte del código       |
| valor#    | Repetición del mismo error en otra parte del código       |
